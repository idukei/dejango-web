Explotación, Desbordamiento de Buffer¶

En el CTF Brainstorm de Tryhackme, despues de conseguir acceso a la maquina, encuentro el archivo chatserver.exe. Este archivo se ejecuta en el servidor y lo explotare creando un desbordamiento de buffer para tener acceso al servidor.

Necesitare la herramienta Inmunitty Debugger, el script Mona y crear algn script para crear un desbordamiento de memoria.

1 - Abrir en chatserver.exe

2 - Enviar caracteres a la aplicacin hasta crear el desbordamiento

python -c 'print "A" * 2500'
